Features

Feature One

걸음걸이 인식은 걸음걸이를 촬영해 보폭과 실루엣, 신체 특징 등으로 사람의 걸음걸이 형태를 분석하고 특징을 추출해 사용자를 인증하는 방법입니다.

보행의 5개의 정적특징 : 어깨 너비, 엉덩이 너비, 골반 길이, 다리 길이, 각 신체 부분의 다른 신체 부분에 대한 비율

보행의 5개의 동적특징 : 허벅지 사이의 각도 변화량, 상체와 골반 밑 부분의 각도 변화량, 허벅지와 종아리 사이의 각도 변화량

위의 2가지의 분류 기법을 사용해서 신원을 파악합니다.

Feature Two

귀 인식은 귀를 촬영해 귀의 특징으로 사람의 귀 형태를 추출해 사용자를 인증하는 방법입니다.

귀 형태 추출 : 전체 영상 중 Haar Cascade Classifier로 귀 형태의 음영차를 이용해 귀 형태를 추출합니다.

사용자 인증 : Convolusion Neural Network 모델을 이용하여 해당 귀를 가진 사람의 신원을 파악합니다.
